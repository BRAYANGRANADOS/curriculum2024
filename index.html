experiencia de trabajo

EduQuiz C++

enero 2012- trabajo

"EduQuiz C++" es una plataforma diseñada para la creación y administración de cuestionarios educativos. Este proyecto utiliza el lenguaje de programación C++ para implementar la lógica de evaluación de las respuestas a las preguntas. La plataforma proporciona herramientas para la creación de preguntas, la asignación de puntajes y la generación de informes. Con un enfoque en la eficiencia y rendimiento de la lógica de evaluación, EduQuiz C++ busca ofrecer una solución efectiva y versátil para la evaluación educativa a través de cuestionarios interactivos.

Sistema de Monitoreo y Control Doméstico

mar 2010 -dec 2011

El "Sistema de Monitoreo y Control Doméstico" es un proyecto que integra Arduino y Python para crear una solución inteligente para el hogar. Utiliza sensores en Arduino para monitorear condiciones como temperatura, humedad y movimiento, y actúa sobre dispositivos mediante actuadores. Python se encarga de la interfaz gráfica de usuario, permitiendo al usuario visualizar datos, controlar dispositivos y recibir alertas. Este proyecto brinda una solución eficiente para la automatización y supervisión inteligente del entorno doméstico.
